# Shopping Cart  
A simple E-commerce website using Flask using voice command
 
## Dependencies
 ##
1. Python3

2. Flask
3 and mysql
4 speech recognization


## How to run
 ##
1. Set up database by running database.py

2. Run flaskapp.py

3. Enter localhost:5000 in the browser.
